Solution 1: Server-side Upload

Use your server to handle uploads with API secret (Recommended)

Create Cloudinary Upload Preset

Steps to Create Upload Preset:

  1. Go to Cloudinary Console
  2. Navigate to Settings → Upload
  3. Scroll down to Upload presets
  4. Click Add upload preset
  5. Set Preset name to: portfolio_unsigned
  6. Set Signing Mode to: Unsigned
  7. Set Folder to: portfolio
  8. Click Save

Alternative: Use Server-side Upload

The server-side method is more secure and doesn't require creating upload presets.
